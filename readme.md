将原神的抽卡系统带入Europa Universalis V

## ✨ 当前实装功能

### 🎲 抽卡系统 ✅ **完整可用**

## 🚀 使用方法

1. **启动游戏**: 在启动器中加载此mod
2. **进入游戏**: 开始或加载游戏
3. **打开角色面板**: 选择您的统治者
4. **点击"祈愿"交互**: 花费金币进行抽卡
   - 单抽: 16金币
   - 十连: 160金币
5. **查看结果**: 
   - 抽到新角色会自动创建并加入您的国家
   - 抽到重复角色会提升命座等级
   - 触发专属故事事件
6. **Debug查看**: 使用游戏内Debug Tool查看：
   - `gacha_pity_count` - 当前5星垫子（0-89）
   - `gacha_pity_4star` - 当前4星垫子（0-9）
   - `gacha_total_rolls` - 总抽卡次数
   - `gacha_is_guaranteed` - 大保底状态
   - `gacha_constellation_lvl` - 命座等级
   - `gacha_block_has_4star` - 当前块是否已出4/5星

## 🔧 技术要点

### 随机数生成
使用质数混合算法确保分布均匀：
```
rand = 937 + 17×total_rolls + |gold| + 13×pity + 7×block_index
```

### Silent内核架构
- 计算与展示分离
- Silent内核只负责逻辑计算
- Event层负责UI展示和玩家交互

### 概率公式
- 5星基础: 0.6% (6/1000)
- 5星软保底: 74抽起每抽+6%
- 5星硬保底: 90抽100%
- 4星基础: 5.1% (51/1000)
- 块内保底: 每10抽强制至少1个4/5星
